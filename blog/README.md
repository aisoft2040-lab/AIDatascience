## üöÄ Learning Paths

### Path 1: Complete Beginner ‚Üí Production AI Engineer (6 months)
1. Complete [Module 0-5](curriculum/README.md) - Foundations and core skills
2. Build [Portfolio Projects 1-3](curriculum/README.md#portfolio-projects)
3. Follow [Production Deployment Tutorial](tutorials/06-deployment.md)
4. Complete [Module 6-8](curriculum/README.md) - Production engineering

### Path 2: ML Engineer ‚Üí AI Engineer (3 months)
1. Review [Module 1-2](curriculum/README.md) - LLM specifics
2. Deep dive [Module 5: RAG](curriculum/README.md#module-5-retrieval-augmented-generation-rag)
3. Focus on [Module 7-8](curriculum/README.md) - Observability and cost
4. Build production system - [Portfolio Project 2](curriculum/README.md#portfolio-projects)

### Path 3: Software Engineer ‚Üí AI Integration Specialist (2 months)
1. Quick overview [Module 1-2](curriculum/README.md)
2. Focus [Module 3-4](curriculum/README.md) - Evaluation and safety
3. Master [Module 6-7](curriculum/README.md) - Deployment and monitoring
4. Explore [Enterprise Integration](curriculum/README.md#module-10-enterprise-integration)
